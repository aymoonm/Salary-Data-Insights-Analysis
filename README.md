# 💰 Salary Data Analysis & Insights

## 📌 Project Overview
This project performs an in-depth **Exploratory Data Analysis (EDA)** on a salary dataset using **Python**. The goal is to identify the key factors that drive salary levels, including professional experience, job titles, and educational backgrounds, to provide data-driven insights into compensation trends.

## 🛠️ Key Tasks Performed
- **Data Cleaning:** Cleaned the dataset and handled missing or inconsistent values to ensure accurate statistical reporting.
- **Demographic Analysis:** Analyzed salary distributions based on gender, age, and education level.
- **Experience vs. Salary:** Visualized the correlation between years of experience and salary growth using **Seaborn** and **Matplotlib**.
- **Job Role Categorization:** Grouped and compared average salaries across various job titles and industries.
- **Statistical Visualization:** Created box plots and scatter plots to identify outliers and salary clusters.

## 🧰 Technologies Used
- **Python**
- **Pandas** (Data Manipulation)
- **Matplotlib & Seaborn** (Data Visualization)
- **Numpy** (Numerical Operations)
- **Jupyter Notebook**

## 📊 Key Insights
- Identified a strong linear correlation between years of professional experience and annual salary.
- Visualized the salary gap (if any) across different educational degrees.
- Highlighted the top-paying job roles within the analyzed dataset.
